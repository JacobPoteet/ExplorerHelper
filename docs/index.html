Everything you need to triage a folder

Previews, filters, and keyboard-first flows — none of it leaves your machine.

🃏

Swipe-to-triage

The fastest way to clear out a folder: review files one at a time like a dating app — swipe or ←/→ to reject or keep, ↓ to skip. Nothing touches the disk until you Commit: a review screen shows both piles so you can swap or unmark any file, then rejects go to the Recycle Bin and keepers can move to a folder of your choice — one Ctrl+Z reverses the whole commit.

🖱️

Explorer context menu

“Clean this folder” on folders and folder backgrounds. Per-user, no admin required.

👁️

Live previews

Images render natively, video and audio play inline, and PDFs open in the built-in viewer.

⏱️

Scrub timeline

Video previews come with play/pause, elapsed and total time, and a draggable slider to jump anywhere in the clip.

📋

Preview details

A strip under the preview surfaces the selected file’s metadata — type, size, resolution, length, frame rate, bit rate, created/modified dates. Toggle each one on or off in Settings.

⚡

Quick rename

Type a name, press Enter, jump to the next file. Collisions auto-number (Clip, Clip 2…) and the extension is preserved.

🏷️

Quick-use buttons

Build a name in one click from your own reusable buttons, plus dynamic buttons for today’s date or the file’s created date — formats configurable in Settings.

⌨️

Keyboard-first

Del to Recycle Bin, F2 rename, Enter open, K/X/U flag — fly through a folder without the mouse.

↩️

One-key undo

Renamed, deleted, or committed the wrong thing? Ctrl+Z puts it back — files come straight out of the Recycle Bin.

🔠

Filter & sort

Filter by name and toggle file types on or off. Sort by name, size, date, or type — folders always first.

🗑️

Safe by default

Deletes go to the Recycle Bin, never a permanent wipe. Multi-select to clear several at once.

🔄

Always up to date

When a new release ships, an update button appears in the toolbar — one click downloads it, installs silently, and restarts the app right where you were. Toggleable in Settings.

From cluttered to clean in four steps

No setup rituals — it lives in the menu you already use.

Right-click a folder

Pick Clean this folder from the Explorer context menu (or the background of an open folder).

Swipe through it

Preview each file and flag it keep or reject — swipe, ←/→, or K/X. Rename a run of clips with Enter along the way.

Commit once

Review both piles, then Commit: rejects go to the Recycle Bin and keepers can move to a folder you choose.

Undo mistakes

Changed your mind? Ctrl+Z reverses the last rename, delete, or the whole commit — no digging required.

Install

Grab the installer, or run from source.

⬇ Installer (recommended)

Download ExplorerHelper-Setup-*.exe from the latest release and run it. It installs per-user (no admin prompt) and registers the context-menu entries. Uninstalling removes them again — and from here the app keeps itself up to date with one-click updates.

Download latest release

A portable zip is published with every release too — use the Add context menu button inside the app.

🛠 Build from source

Requires the .NET SDK 8 or newer.

# Debug run
dotnet run --project src/ExplorerHelper

# Publish self-contained exe + portable zip
./build.ps1 -Version 0.4.0
Windows 11 note: the entry appears under Show more options (the classic menu), or immediately when you Shift+right-click. For a top-level entry (no “Show more options” detour), build the sparse MSIX package — see packaging/README.md.
